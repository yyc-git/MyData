# “人生游戏”说明书～如何成高手？如何通关？ - 杨元超

> 日期: 2020-10-16 15:40
> 源: https://www.cnblogs.com/chaogex/p/13826798.html

我们的人生真的就是一场游戏，根本就是一场梦。不管游戏输赢，不管我们如何游戏，都不会影响我们的神圣本体，也就是合一的一体自性～所谓一场梦境一场空。从梦中醒来后，梦中的一切都消失了，不会影响醒来的那个存在。


# 游戏的开始


一开始，万物都是一体的，我们都是一个，完全的和谐、喜悦，充满了无条件的爱。


到某个时候，合一本体的一部分提出了一个问题：如果我们不是一体的，会怎样？如果有分裂，会怎样？

有了问题，自然就有了答案。合一本体的一部分开始创造这个“分裂”的体验，从而去体验：与一体分离是怎样的体验？

然而“分裂”是不可能发生的，因为合一本体就是一切，除此之外没有其它！

所以只能进入幻象，开始一场梦。合一本体的一部分让自己相信“分裂”的信念，也就是我们以为自己与神圣一体的本体能够分离的时候（当然这从来没有发生过，分裂只是幻象），这部分在幻象中就从合一本体“分裂”了，变成了一个个体，开始了这场“分裂”的游戏；


在某个时候，我们决定在“分裂”游戏中创造一个小游戏，叫做“人生”。

出生的时候，我们从“分裂”的大游戏，进入了“人生”这个小游戏；

死亡的时候，我们退出了“人生”这个小游戏，进入“分裂”的大游戏中；

我们会不断地轮回，也就是不断地进入“人生”这个小游戏。每次都会前进一点，直到我们完成“人生游戏”的三个阶段，从“人生游戏”中毕业，然后选择进入“分裂”大游戏中其它的小游戏玩耍；或者回归合一本体休息，休息够了再来选择玩“分裂”大游戏或者其它的大游戏。


什么是“人生游戏”的三个阶段？


# 第一个阶段


第一个阶段叫遗忘。在这个阶段，我们越来越遗忘自己是谁，也就是遗忘自己与万物合一的一体性，把一切都看成与自己无关，分裂。

大部分人都处于这个阶段。人们不断地评判，把“身体”和“个人身份”当成了自己的全部，不断地下降。


当一个人下降到谷底时，可能就长期一直呆在那。直到自己忍无可忍（在第一阶段，人经常处于痛苦、沮丧、怨天尤人的状态），或是接受合一本体的其它部分的提醒而觉醒（合一本体一直在与我们沟通，只是在第一阶段的人一直没静下心来聆听它），从而进入第二阶段。


# 第二个阶段


第二个阶段叫忆起。在这个阶段，我们开始回忆起自己是谁，为什么来这里，要去哪里。开始忆起自己与万物是一体的。


人们在这个阶段会开始给予，服务自己和众生。因为明白自己与万物是一体的，所以服务他人，就是服务自己。所以我们不断地去给予，从而不断地加倍接受到自己给出的一切。因此，生活越来越轻松自在，我们体验到心想事成。喜悦、和平、爱是这个阶段的主流。


当一个人在无尽地服务他人和服务自己后，如果厌倦了这一切，那么就可以进入第三阶段。


# 第三个阶段


第三个阶段叫放下。放下一切，因为一切都已经体验过了。所有的角色我都已经扮演过了，所有的人生戏剧我都经历过了，一切都够了～圆满了～


放下一切，回归平衡，我们开始退出人生游戏，回归合一的本体。


在这个阶段，我们是一体的创造。意思是我们站在合一本体的角度，一起进行创造。这种创造具有无限的创造力，因为我们与神同在！我们就是神！


# 如何成“人生游戏”的高手


成为高手后，虽然还在“人生游戏”中，但已经可以心想事成，一切都轻而易举了～此时已经在第二阶段，与合一本体的洪流一致，顺流而下，整个宇宙都在服务我。所谓我为人人，人人为我。


因此，当我们从第一阶段进入到第二阶段时，就开始成为高手了，开始体验到“人生游戏”真是越来越好玩了！


如何才能成为高手，不断地有更多的乐趣？给予～服务他人～


给出去一分，就会收到三分甚至十分的回报！真是一本万利！

而且死后也可以带走！死前给出的一切，只要还没有变成果，都会被你带到死后，甚至下辈子～


所以说，我们去无限地给予吧～因为我们是无限的～越是给予，越是收获更多～资源是无限的～因为资源只是一个“理念”而已，也就是一个念头而已。当我们分享“理念”，分享一个念头时，只会让这个念头变得更强大！所以说越是给予，越是获得更多～因为一切都是念头～去分享念头吧～把爱带给世界，从而收到更多的爱～


# 如何从“人生游戏”中通关


通了关后，我们就可以自由选择是否再来玩一次“人生游戏”；或者玩“分裂”大游戏中其它的小游戏；或者继续回归合一本体，从而再从更大的“分裂”大游戏中通关，去玩其它的大游戏！


如何从“人生游戏”中通关？开始放下，从第二阶段进入第三阶段。放下一切，让一切的因果平衡～与合一本体合作，一体创造～


当我们想要体验的都体验够了，自然就想要通关了；

当一切都平衡了，自然就通关了。
