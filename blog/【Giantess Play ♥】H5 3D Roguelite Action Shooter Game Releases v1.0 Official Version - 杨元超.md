# Game Introduction


**【Giantess Play】** is a roguelite H5 3D giantess action shooter game, playable on both mobile and PC.


Developed for over a year with more than 30 versions released, the game boasts over 100,000 effective lines of code.


The recent v1.0 official release is characterized by **stability, mature core gameplay, free-to-play access, and an optional activation feature.**

Activating the game provides the following benefits:


- Unlock **All Giantess Characters** (after capture)

- Gems Obtained ×150%

- Chance to Obtain Artifacts, Revival Stones, and Capture Devices ×200%


**[Activate the Game Here](https://www.patreon.com/GiantessPlay)** - Activation price: 7$/month

# Game Features


- **Theme:** Anime-style beautiful giant giantess

- **Gameplay:** Roguelite random battles, diverse playstyle combinations, player progression, dynamic difficulty

- **Bosses:** Challenge various giantess bosses

- **Dual Roles:** Switch between Tiny characters (shooting-focused) and Giantess characters (action-focused)

- **Platform:** H5 3D web game – play instantly online without downloads, runs on all platforms (mobile, PC, etc.)
